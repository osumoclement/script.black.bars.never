# addon.black.bars.never
An addon to automatically remove black bars in KODI

This addon eliminates the problem of black bars. If the black bars are hardcoded, the addon will automatically 
		detect this and remove them. If the video is a wide screen format, the addon will also detect this and remove the black bars too.
    
Core functionality is adapted from
https://github.com/drthaipham/CIHAutoNox
